		### CHF SCHEDULING PROGRAM ###
		
##################################################################
#####  Steps to Set up the environment and do the first run  #####	
##################################################################

1. Download Visual Studio Community 2019 for free by 
   copying and pasting the following URL into your browser
   
	https://download.visualstudio.microsoft.com/download/pr/5e397ebe-38b2-4e18-a187-ac313d07332a/169156e6e9a005d49b357c42240184dc1e3ccc28ebc777e70d49257c074f77e8/vs_Community.exe

2. Use your Microsoft account to log in and install 
   the following Windows Workloads: 

	- Desktop Development with C++
   	- Universal Windows Platform development

3. Download the JSON Library for free by
   copying and pasting the following URL into your browser

	https://codeload.github.com/nlohmann/json/zip/develop

4. Open the zip and search for the file "json.hpp".
   Then, copy it and paste it onto the following path

	C:\Program Files (x86)\Microsoft Visual Studio\2019\Community\VC\Tools\MSVC\14.25.28610\include

5. Search for the file "Project X.vcxproj" int the Program's
   folder and open it with Visual Studio 2019 (Location below)

	GIT-Project-X-master\Project X\Project X\Project X.vcxproj

6. Once the project starts, tap on the green Play button that says
   Local Windows Debugger to run the pogram.

7. Type the number of the month and the year, then press any key to 
   close the window.

8. In the same folder where you found "Project X.vcxproj" there will 
   be the updated version of your Final Schedule. A shortcut has been
   created and you can now cut it and paste it onto your desktop for 
   easy access. To do so, look for the file "Final Schedule" with 
   type of file "Shortcut" and drag it to your Desktop.




################################################################
#####  Steps to Run the program once everything is set up  #####	
################################################################


1. Open the file "Project X" in the "Final Schedule" folder

2. Once the project starts, tap on the green Play button that says
   Local Windows Debugger to run the pogram.

3. Enter the desired Month and Year for your new Schedule and press
   any key to close the window

4. Open the file "Final Schedule" in the "Final Schedule" folder



################################################################
#############  Steps to Run the FinalSchedule.exe  #############	
################################################################

1. Double click on the file "FinalSchedule.exe"

2. Enter the desired Month and Year for your new Schedule and press
   any key to close the window

4. Open the file "Final Schedule" in the "Final Schedule" folder
